# Guía de usuario

## Layout Pearl SureType

Teclado simétrico de **5 columnas**, centrado en pantalla con teclas grandes (ideal para dedos gruesos):

```
 Q/W   E/R   T/Y   U/I   O/P
 A/S   D/F   G/H   J/K    L
 Z/X   C/V   B/N    M    del
alt   sym  space   aA   enter
```

## Escribir con SureType

Cada tecla tiene **dos letras**. Pulsa una vez — el motor predictivo elige la letra correcta según la palabra que estás formando.

- No necesitas tocar mitad izquierda/derecha
- Las **sugerencias** aparecen en la barra superior
- Toca una sugerencia para aceptarla

## Modo símbolos

1. Pulsa **sym**
2. Las teclas muestran números y símbolos (`!`, `1`, `@`, `0 +`, `#`…)
3. Pulsa **ABC** para volver a letras

## Teclas especiales

| Tecla | Función |
|-------|---------|
| del | Borrar |
| space | Espacio (confirma palabra y aprende) |
| aA | Mayúsculas / bloqueo |
| enter | Confirmar línea |
| alt | Capa alternativa de símbolos |

## Aprendizaje de palabras

Al pulsar espacio o aceptar una sugerencia, la palabra se guarda en tu diccionario local. Cuanto más escribes, mejores son las predicciones.

Puedes desactivar el aprendizaje o borrar el diccionario desde la app de configuración.
